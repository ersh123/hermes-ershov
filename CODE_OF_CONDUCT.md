# Code of Conduct

This project is for calm, useful, no-drama collaboration.

## Expected behavior

- Be respectful.
- Be direct without being abusive.
- Keep feedback useful and specific.
- Assume good intent unless the evidence says otherwise.
- Respect privacy, boundaries, and the repo's safety rules.

## Unacceptable behavior

- harassment, slurs, threats, or intimidation
- stalking, doxxing, or sharing private data
- deliberate disruption, spam, or sabotage
- trying to leak secrets or bypass writeback safety
- dogpiling people over a bug report or review comment

## Enforcement

Maintainers may remove comments, close issues, reject pull requests, and restrict access when needed.
For serious privacy or safety issues, use the private security reporting path instead of a public thread.
